# weel 3 실습 기록

## 목표
pygame를 실행하고 AI를 이용해 키보드 입력 구현

## AI 대화 기록

**Q1: 위 코드에서 키보드 입력을 받아 공을 움직이게 하는 방법을 알려줘**

- AI답변:  pygame.key.get_pressed()를 이용해 키입력을 받고 누르는 키에 따라 x, y값을 증감시킴
- 적용 결과: 성공

**Q2: 공이 화면 밖으로 나가는 것을 막는 방법을 알려줘**

- AI답변: max와 min을 이용해 x와 y값을 제한함
- 적용 결과: 성공

**Q3: Shift를 누르면 대쉬하는 기능을 만드는 법을 알려줘**

- AI답변: KEYDOWN을 이용해 입력을 받고 이동하는 방향에 따라 좌표를 크게 이동시킴
- 시행착오: KEYDOWN이 무엇인지 몰라서 다시 질문
- 해결: get_pressed()와 달리 KEYDOWN은 키를 누르는 순간 딱 한번만 입력 처리함
- 적용결과: 성

## 배운점

- get_pressed()를 이용하면 키 입력을 받을 수 있다
- max와 min을 이용해 x와 y값을 제한할 수 있다
- KEYDOWN을 이용하면 키를 누르는 순간 한번만 입력 처리가 가능하다
