# 4주차 실습 정리

## 오늘 한 것
 - AABB / 원형 / OBB Bounding Box 시각화
 - 세 방식의 충돌 판정 차이 관찰

## 관찰 내용

### AABB vs 원형
 - 원형은 모서리 쪽에서 AABB보다 충돌 범위가 작아서, 모서리로 접근할 때 AABB가 먼저 충돌 판정된다.
 - 원형의 지름이 AABB읜 가로,세로 길이보다 길면 상하좌우로 접근할 때 원형이 먼저 충돌 판정된다.

### AABB vs OBB
 - 오브젝트가 회전하지 않을 때는 AABB와 OBB가 동일하게 동작한다.
 - AABB는 오브젝트의 회전과 상관없이 충돌 판정이 그대로이다.
 - OBB는 오브젝트가 회전하면 충돌 판정이 같이 회전한다.

## AI와의 대화에서 배운 것
 - 거리를 비교할 때 math.sqrt를 생략하고 거리의 제곱끼리 비교해도 결과가 같다는 것을 배웠다.
 - 두 오브젝트 사이에 분리축이 존재하는지 검사하는 충돌 감지 알고리즘인 SAT에 대해 배웠다.

## 내 게임에 적용한다면
 - 세 방식 중 OBB를 메인으로 사용할 것이다.
 - 회전하는 오브젝트가 있을 경우 AABB는 알맞지 않다.
 - 원형은 구현이 단순하지만 정사각형 오브젝트에는 맞지 않다.
